# Chronos

Chronos is a time-blocked productivity system designed for students.
It combines daily scheduling, goal tracking, habit building, and pomodoro-based focus
into a single, offline-first application.

## Core Principles
- Time blocking over task lists
- Goals drive daily schedules
- Habits are built through small, repeatable actions
- Deep focus via pomodoro tracking
- Privacy-first and offline-capable

## Tech Stack
- Backend: Rust + Axum
- Database: PostgreSQL
- Frontend: React + TypeScript (PWA)
- Infra: Docker, GitHub Actions
